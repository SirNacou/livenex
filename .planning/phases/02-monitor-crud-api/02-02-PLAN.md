- `DELETE /api/monitors/:id` → 204 no body; GET after → 404
  </what-built>
  <how-to-verify>
Start the dev server (ensure Docker PostgreSQL is running):
```
bun run dev
```

Then run these curl commands (or use the httpie equivalents):

**1. List monitors (seed data should appear):**
```
curl -s http://localhost:3000/api/monitors | jq .
```
Expected: array with at least the two seed monitors (http + ping)

**2. Create an HTTP monitor:**
```
curl -s -X POST http://localhost:3000/api/monitors \
  -H "Content-Type: application/json" \
  -d '{"type":"http","name":"Test HTTP","url":"https://httpbin.org/get","interval":30}' | jq .
```
Expected: 201, full monitor object with `id`, `createdAt`, `currentStatus: "pending"`

**3. Create a ping monitor:**
```
curl -s -X POST http://localhost:3000/api/monitors \
  -H "Content-Type: application/json" \
  -d '{"type":"ping","name":"Test Ping","host":"1.1.1.1","interval":60}' | jq .
```
Expected: 201, full monitor object with `host: "1.1.1.1"`, `url: null`

**4. Get the HTTP monitor by ID (use the ID from step 2):**
```
curl -s http://localhost:3000/api/monitors/<ID_FROM_STEP_2> | jq .
```
Expected: 200, same object

**5. Update the HTTP monitor's interval:**
```
curl -s -X PUT http://localhost:3000/api/monitors/<ID_FROM_STEP_2> \
  -H "Content-Type: application/json" \
  -d '{"interval":45}' | jq .
```
Expected: 200, monitor with `interval: 45`

**6. Toggle enabled state:**
```
curl -s -X PATCH http://localhost:3000/api/monitors/<ID_FROM_STEP_2>/toggle | jq .
```
Expected: 200, monitor with `enabled: false` (was true)

**7. Delete the ping monitor (use ID from step 3):**
```
curl -s -X DELETE http://localhost:3000/api/monitors/<ID_FROM_STEP_3> -o /dev/null -w "%{http_code}"
```
Expected: 204

**8. Confirm deletion:**
```
curl -s http://localhost:3000/api/monitors/<ID_FROM_STEP_3> | jq .
```
Expected: 404 with `{"error": "Monitor not found"}`

**9. Validation rejection — interval too low:**
```
curl -s -X POST http://localhost:3000/api/monitors \
  -H "Content-Type: application/json" \
  -d '{"type":"http","name":"Bad","url":"https://example.com","interval":5}' -o /dev/null -w "%{http_code}"
```
Expected: 400 (validation error, interval minimum is 10)
